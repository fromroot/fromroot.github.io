#깃헙 블로그 10분안에 완성하기

1. 사용할 테마\( minimal-mistakes \) 선택하기
2. 테마 fork하기: 자신의 repository에 프로젝트 복제
3. _config.yml 파일의 url을 자신의 repository url로 변경:
   자신의 블로그 url 접속 시 적용된 테마 확인가능
4. 자신의 블로그에 새로운 포스팅 생성:
   영상 속 url 사이트 상의 레이아웃 복붙하여 title 수정
5. markdown 에디터 사용:
   jupyter notebook
6. 테마 변경 가능:
   _config.yml에서 수
